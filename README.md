# README

Work in progres...

A template for creating Modular Monoliths with C#.

Heavily inspired by:
- https://github.com/kgrzybek/modular-monolith-with-ddd

## Development

Test the package:

```bash
dotnet new install working/content/modulith
```

Package:

```bash
cd working
dotnet pack
dotnet new install bin/Release/Modulith.0.1.0.nupkg
```
